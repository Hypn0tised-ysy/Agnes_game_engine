# 架构     
1. 输入管理模块  
负责获取用户输入    
2. 策略模块  
负责执行一些策略  
3. 场景管理模块  
管理、更新场景   
4. 渲染模块   
执行渲染和画面输出  
5. 音频音效模块  
管理声音、混音、播放  
6. 网络通信模块   
管理网络通信  
7. 文件IO模块  
管理资源加载和参数保存恢复    
8. 内存管理模块  
调度内存资源  
9. 驱动模块  
根据时间、事件驱动其他模块    
10. 辅助模块  
执行调试、log输出等功能  
11. 应用程序模块    
抽象处理配置文件，平台通知，创建窗口等与平台对接的部分  

# 源代码说明
## Common
不同平台的通用代码   
## Interface  
定义各种接口:  
如不同平台的应用程序接口  
运行时模块的接口   
## Empty  
定义一个空平台
